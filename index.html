Tanpa Template

Portofolio ini murni hasil karya saya dan design saya sendiri. Pembuatan portofolio ini menggunakan framework bootstrap 3.
Untuk memperindah website dengan animasi (seperti loader di awal), saya mencari referensi di internet dan mengimplementasikannya pada portofolio ini.

Dengan Template dari BootstrapMade

Portofolio ini menggunakan template yang sudah ada di internet. Pembuatan portofolio ini hanya perlu memodifikasi tata letak dan konten pada website sesuai dengan apa yang di perlukan.
Pada pembuatan website ini, tentunya saya menambahkan material-material tertentu untuk mempercantik tampilannya, seperti penambahan animasi loading di awal.
