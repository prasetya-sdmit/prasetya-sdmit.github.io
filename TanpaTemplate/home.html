Mobile App







Aplikasi ini dibuat saat saya akan mengikuti kompetisi IT Days yang diadakan di kampus. Aplikasi ini memiliki fungsi yang hampir sama dengan Go-Jek atau Grab, hanya saja aplikasi ini dikhususkan untuk layanan sepatu seperti mencuci, mengecat ulang, memperbaiki dan sol sepatu. Aplikasi ini menggunakan bahasa pemrograman Java dan dibuat dengan menggunakan alat Android Studio. Aplikasi ini masih memiliki banyak kekurangan, seperti tidak terkoneksi secara otomatis antara pelanggan dan mitra.

Web




Sistem Pendukung Keputusan berbasis Web
Aplikasi ini dibuat pada saat penulisan skripsi. Aplikasi ini dapat digunakan untuk menentukan pegawai terbaik berdasarkan kriteria yang telah ditentukan. Aplikasi ini dibuat dengan menggunakan bahasa pemrograman PHP, HTML dan CSS. Karena kemampuan saya yang terbatas, saya memerlukan bantuan untuk membangun aplikasi seperti googling untuk mencari solusi masalah dan template yang tepat agar mendapatkan hasil yang maksimal. Tentunya dari semua template yang saya dapatkan, saya perlu memodifikasinya sedemikian rupa agar aplikasi dibuat sesuai dengan kebutuhan dan tujuan.
